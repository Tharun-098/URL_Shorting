# URL Shortener

A full-stack **URL Shortening Service** that takes long URLs and creates short, easy-to-share links. This project is built with **Express, MongoDB, and React**, and is deployed on **Vercel**.

ðŸ”— Live Demo: https://url-shorting-jdip.vercel.app/

## Features

- Create shortened URLs
- Redirect to original URL
- Click tracking and analytics
- Clean and responsive UI

## Tech Stack

- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Frontend:** React
- **Deployment:** Vercel

## Get Started

1. Clone the repository  
2. Install dependencies (`npm install`)  
3. Setup environment variables  
4. Run the app locally
